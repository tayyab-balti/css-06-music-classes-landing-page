# Delivery Request Form UI (HTML/CSS Project)

A professional and detailed web form UI designed for collecting delivery and service requests. This project demonstrates proficiency in structuring complex forms and handling various input types, including:
- **Two-Part Address Input:** Separate sections for Delivery Address and Pickup Location, ensuring data clarity.
- **Diverse Input Fields:** Utilization of text inputs, drop-downs (`<select>`), and radio buttons for user options.
- **Complex Layout:** Organized grouping of related fields (Customer Info, Addresses, Item Details) for enhanced user experience.
- **Conditional Service Options:** Radio buttons for "Assembly Required," "Any stairs?," and "Move Existing Furniture?"
- **Interactive Element Placeholder:** A mock "Bill To" section with a drag-and-drop placeholder.

Features a **fully responsive and clean layout** built using pure HTML5 and CSS3, demonstrating strong **Form Styling** and structural design.
